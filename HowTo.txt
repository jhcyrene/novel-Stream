StoryStream Application – Setup & Explanation

Part 1: Setting up the Project

Step 1: Create a directory Create your main project folder (e.g., StoryStream).

Step 2: Create a Virtual Environment using bash or terminal

  python -m venv venv

Step 3: Activate Virtual Environment use terminal

  Windows: venv\Scripts\activate
  Mac/Linux: source venv/bin/activate

  or
  Windows: venv/Scripts/activate.bat

Step 4: Install Requirements (see the Requirements or requireCmd for Requirements ) 
  example 
    pip install flask flask_sqlalchemy flask_wtf werkzeug

Step 5: Run the Flask (terminal)
  py app.py
